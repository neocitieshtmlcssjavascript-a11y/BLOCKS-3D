# DynaBlocks (Roblox 2004 Prototype) on Python 🧱

Реплика исторического прототипа платформы **Roblox** 2004 года, когда она еще носила название **DynaBlocks**. Проект воссоздает атмосферу ранней альфа-версии с физикой падающих блоков, разрушаемостью и ретро-интерфейсом.

## ✨ Особенности
*   **Ретро-стилистика**: Оригинальная цветовая палитра 2004 года (классическое голубое небо, зеленая земля и желтые строительные блоки).
*   **Физика гравитации**: Блоки падают вниз и реалистично укладываются друг на друга.
*   **Сетка (Grid Snapping)**: Автоматическое выравнивание блоков при создании для ровного строительства.
*   **Интерактивность**: Быстрое создание и мгновенное разрушение объектов.

## 🚀 Быстрый старт

### Требования
Для запуска игры вам понадобится установленный **Python 3.x** и библиотека **Pygame**.

### 1. Установка зависимостей
Откройте терминал (или командную строку) и установите движок Pygame:
```bash
pip install pygame
```

### 2. Запуск игры
Перейдите в папку с файлом `dynablocks.py` и выполните команду:
```bash
python dynablocks.py
```

## 🎮 Управление
*   🖱️ **Левая кнопка мыши (ЛКМ)** — Создать (заспавнить) блок в небе.
*   🖱️ **Правая кнопка мыши (ПКМ)** — Удалить (разрушить) блок на экране.
*   ❌ **Крестик окна** — Выход из симулятора.

## 🛠️ Технологический стек
*   **Язык**: Python
*   **Движок**: Pygame (2D Графика и обработка физических коллизий)
